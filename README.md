# AdbController
Android adb commands controller.<p>

IDE: Eclipse<p>
Architecture: MVC<p>
Design pattern: Command pattern<p>
Java GUI framework: JavaFX<p>
Logger lib: log4j<p>
JDK: it must be used with Java 8 or above<p>
